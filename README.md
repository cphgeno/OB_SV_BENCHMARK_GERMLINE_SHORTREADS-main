# OB_SV_BENCHMARK_GERMLINE_SHORTREADS-main
An Omnibenchmark workflow running multiple structural-variant (SV) callers on short-reads data and benchmarking their results against a single truth set with Truvari

This repository contain an adaptation of the snakemake pipeline in https://github.com/cphgeno/SV-Benchmarking-Germline into Omnibenchmark.
